backlog.d/047-replace-real-repo-saturation-probe.md

Replace brittle whole-workspace one-shot freeze probe with bounded review-context probe.

Outcome: `arena-freeze arenas/pr-review-correctness-v0` can produce a non-inconclusive probe verdict before candidate search, without treating context overflow or empty model output as evidence.

Chosen Design

Keep probe as `kind = "oneshot"` reference, but add manifest-driven workspace modes. Default remains full workspace; freeze probe uses a bounded review-context prompt: task instruction, `PR.diff`, intent, README if present, changed-file snippets, and manifest limits.

Quality System

Live oracle: `arena-freeze` + `arena-validate --probe-run`. Structural gate: Rust tests + `bin/gate`. Critic: artifact-only review of diff and 047 oracle.

Standards

No scorer weakening. Probe remains reference-only and excluded from search/recommendations. Candidates still never see `tests/` or `solution/`.

Acceptance

Correctness v0.3 freeze yields `probe_verdict` saturated or unsaturated, not inconclusive from overflow/empty content. Validation report explains semantics.

Stop If

Live provider errors persist after bounded prompt, model catalog/auth is missing, or probe result shows true saturation.

Verification System

  • Claim: real-repo arena freeze probe is falsifiable and bounded.
  • Falsifier: probe trials error on context overflow/empty content while validation passes or no useful artifact records why.
  • Driver: `cargo run --quiet --bin threshold -- arena-freeze arenas/pr-review-correctness-v0 --out-dir runs/<047-freeze>`.
  • Grader: `arena-validate --probe-run` status plus `summary.json` probe errors/trials/verdict.
  • Evidence packet: freeze dir with `trials.jsonl`, `summary.json`, and `freeze-report.md`.
  • Cadence: tests before live spend, live freeze before PR.

Alternatives

Raise context window: rejected, still fails on empty provider output and expensive prompts. Chunked multi-call probe: better later, but changes scoring semantics. Tool-using agent baseline: useful but no longer no-tool saturation probe. Bounded review-context one-shot is smallest durable replacement.
