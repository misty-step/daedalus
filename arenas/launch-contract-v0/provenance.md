# launch-contract-v0 provenance

Prepared 2026-06-12 for ticket 030 as the second Threshold task family.

Authoring rule: every defect is synthetic and lives in the visible
`environment/` contract artifacts. Answer keys use broad line spans because
the task is meant to evaluate whether an agent identifies the contract
violation and category, not whether it cites a narrowly calibrated line.

Frozen surfaces for v0.1.0: task fixtures, answer keys, template, split
membership, and scorer constants. Any answer-key or template change requires
an arena version bump and rerun of oracle/null/probe baselines before scores
are compared.
